BitAdminCore介绍

BitAdminCore是基于jQuery，Bootstrap，AdminLTE 的 NET Core开发框架。

架构思想：

定位管理应用程序快速开发能力，以程序易读性为核心。

采用布局与逻辑分离、前端后端分离的思路，降低耦合度。

规范约定：

使用EF Core与SQL混合开发模式；数据库、javascript变量，使用骆驼命名法，例：userName。

BitAdminCore套件

querySuite：

querySuite是查询条件、查询操作、数据展示、分页控制一体化标注式查询套件。支持拖动排序？

formSuite：

formSuite是新增、修改一体化标注式表单套件。

fileupload：

fileupload是一个附件上传、保存、预览一体化套件。

其它组件：

bitSelect、bitRadio、bitCheckbox、bitAutoComplete、bitAutoComSelect

bitTree、picker、datePicker、linkageSelect、treeTable
